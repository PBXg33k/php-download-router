# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.3.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

Please open a new Issue on this repository with a clear description of the Vulnerability.

Adding a Step by Step instruction to reproduce it will make it easier for the developers to address the vulnerability.
